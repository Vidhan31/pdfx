## Libraries used

- Apache Commons IO
- Apache PDFBox
- JavaFX
- Ikonli Fonts
- CSS Themes by AtlantaFX

## Requirements

While the project may work with old versions with workarounds, but for smooth sailing I personally use and recommend:

- Java 17+
- JavaFX 20+
- Scenebuilder
- Maven
- Intellij IDE

## Build the project

- Execute `mvn compile` command.
